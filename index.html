🎙️ WebASR Core

瀏覽器端語音處理測試平台

📌 請確保已啟動測試伺服器：
在專案根目錄執行 python serve.py

📦 本地開發測試

使用本地編譯的 dist 檔案進行測試。適合開發階段的功能驗證和調試。

dist/web-asr-core.bundle.js 本地模型 開發模式

🌐 CDN 整合測試

測試 npm 發布後的 CDN 載入方式。驗證 ULTIMATE 版本的整合功能。

unpkg CDN ULTIMATE 版本 生產模式

📥 NPM 安裝測試

測試透過 npm install 安裝套件的使用方式。需要使用 Vite 或其他打包工具。

npm install ES 模組 Vite 開發

核心功能

  • VAD (Voice Activity Detection) - 語音活動檢測
  • Wake Word Detection - 喚醒詞檢測 (Hey Jarvis, Alexa, Hey Mycroft)
  • Speech Recognition - 語音識別 (Whisper)
  • AudioWorklet 支援 - 低延遲音訊處理
  • WebGPU 加速 - GPU 加速推理

瀏覽器支援

  • Chrome/Edge (完整支援)
  • Firefox (部分功能)
  • Safari (實驗性支援)
